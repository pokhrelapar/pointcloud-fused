# Environment Setup

This project uses a Conda environment defined in `env.yml`.  
Follow the steps below to install and activate it.

---

## 1. Install Conda

You can use either Anaconda or Miniconda.

Download and install one of the following:

- Miniconda: https://docs.conda.io/en/latest/miniconda.html
- Anaconda: https://www.anaconda.com/download

---

## 2. Create the Environment

Open a terminal and navigate to the folder that contains `env.yml`.

Run

```bash
conda env create -f env.yml
```
